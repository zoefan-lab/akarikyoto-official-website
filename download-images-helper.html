🖼️ Image Download Helper

Extract all images from AKARI KYOTO SHIJO React App

📋 How to Use This Tool

  1. Open your React app in Figma Make (the preview pane)
  2. Press F12 to open Developer Tools
  3. Go to the Console tab
  4. Copy and paste the code below into the console
  5. Press Enter
  6. The script will list all images with download links
  7. Right-click each image → "Save Image As..." with the correct filename
⚠️ Important! This tool helps you identify and download images. You'll still need to right-click and save each one manually with the correct filename.

🔧 Console Script

Copy this entire script and paste it into your browser's console (F12 → Console tab):

// Find all images on the page
const images = document.querySelectorAll('img');
console.log(`Found ${images.length} images on the page`);

// Create a list of images with their sources
const imageList = [];
images.forEach((img, index) => {
    if (img.src && !img.src.includes('data:image')) {
        imageList.push({
            index: index + 1,
            src: img.src,
            alt: img.alt || 'No description',
            width: img.naturalWidth,
            height: img.naturalHeight
        });
    }
});

// Print the list
console.log('\n📸 IMAGE LIST:\n');
imageList.forEach(img => {
    console.log(`${img.index}. ${img.alt}`);
    console.log(`   URL: ${img.src}`);
    console.log(`   Size: ${img.width} x ${img.height}px`);
    console.log('');
});

console.log('💡 TIP: Right-click the URL above and select "Open in new tab" to view and download each image!');
